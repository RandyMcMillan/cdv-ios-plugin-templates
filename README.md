#Cordova iOS Plugin Templates



##Installation:  
    $ git clone https://github.com/RandyMcMillan/cdv-ios-plugin-templates.git
    $ git submodules init
    $ git submodules update
  
    
##Usage:   


###Commands:

###`$ ./installer install`
    
###`$ ./installer uninstall`
    
###`$ ./installer upgrade`
    
###`$ ./installer help`

###`$ ./installer readme`
