#Cordova iOS Xcode Plugin Templates




###Installation:

###`$ git clone https://github.com/RandyMcMillan/cdv-ios-plugin-templates.git`

###`$ ./installer install`


###Commands:

###`$ ./installer install`

###`$ ./installer uninstall`

###`$ ./installer upgrade`

###`$ ./installer help`

###`$ ./installer readme`
